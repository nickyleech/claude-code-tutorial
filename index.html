Getting Started

Everything you need to begin

What You'll Need

  • A computer (Windows, Mac, or Linux)
  • An internet connection
  • A project folder (even an empty one!)

Installation

  1. Open Terminal (press Cmd + Space, type "Terminal")
  2. Type this command:
    npm install -g @anthropic-ai/claude-code
  3. Press Enter and wait for installation
  1. Open Command Prompt (press Windows + R, type "cmd")
  2. Type this command:
    npm install -g @anthropic-ai/claude-code
  3. Press Enter and wait for installation
  1. Open Terminal (usually Ctrl + Alt + T)
  2. Type this command:
    npm install -g @anthropic-ai/claude-code
  3. Press Enter and wait for installation

Starting Claude Code

  1. Navigate to your project folder
  2. Type: claude
  3. Look for the welcome message!

Your First Chat

Starting your conversation with Claude

Beginner-Friendly Questions

Get to know your project:

"What does this project do?"

Explore the structure:

"Show me the main files"

"What technologies does this project use?"

Best Practices

How to get the best results

‚úÖ Be Specific

Bad: "Change the text"

‚úÖ Use Plain Language

Good: "Make the button bigger and more colourful"

‚úÖ Break Big Tasks Down
